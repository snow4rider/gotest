**Master Go programming through Test-Driven Development (TDD)**

## 📖 About This Book

This is a free, hands-on guide that teaches you Go by writing tests. Rather than passively reading about the language, you'll explore Go's features by building real code with test-driven development from day one.

**Book URL:** https://quii.gitbook.io/learn-go-with-tests

## 🎯 What You'll Learn

- **Test-Driven Development** fundamentals in Go
- Go language syntax and core concepts through practical exercises
- How to write robust, well-tested Go systems
- Built-in testing capabilities in Go
- Confidence to build production-ready applications

## 👥 Who This Is For

This book is perfect for:

- **New Gophers**: Developers curious about Go who want a structured learning path
- **Testing Enthusiasts**: Go developers who want to deepen their testing knowledge
- **TDD Practitioners**: Programmers who want to learn Go through test-first development

## ✅ Prerequisites

Before starting, you should have:

- A computer with Go installed
- A text editor or IDE
- Basic programming knowledge (variables, functions, `if` statements, etc.)
- Comfort with using the terminal/command line

## 🚀 Why This Approach Works

The author tried several teaching methods with development teams:

**❌ What Didn't Work:**
- Reading technical books with exercises (too slow, low completion rate)
- Code katas (too narrow in scope, inconsistent skill levels)

**✅ What Did Work:**
- Interactive exploration of fundamentals with examples
- Building solid foundations before tackling complex systems
- Learning by doing rather than passive reading

This book combines the best of both worlds: structured learning through tests that document what you've learned while verifying correctness.

## 📚 How to Use This Book

1. **Work Through Sequentially**: The chapters build on each other
2. **Write the Tests**: Don't just read them—type them out yourself
3. **Run the Code**: See the tests fail, then make them pass (Red-Green-Refactor)
4. **Experiment**: Modify the examples to deepen your understanding
5. **Take Notes**: The tests serve as documentation, but add your own insights

## 🛠️ Getting Started

### Install Go

Download and install Go from [golang.org](https://golang.org/dl/)

Verify installation:
```bash
go version
```

### Clone Examples (Optional)

If you want to reference the complete code:
```bash

cd learn-go-with-tests
```

Begin at the introduction and follow along:
